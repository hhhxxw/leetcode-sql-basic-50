# 题目概述

**输入数据：**

一张名为 `Logs` 的表：

- `id`: 自增的主键（通常是连续的整数）。
- `num`: 具体的数字。

**任务目标：**

找出所有 **连续出现至少三次** 的数字。

**示例：**

|   |   |
|---|---|
|**id**|**num**|
|1|1|
|2|1|
|3|1|
|4|2|
|5|1|

**输出：**`1`

（因为 `1` 在 id 1, 2, 3 连续出现了三次。虽然 id 5 也是 1，但它不连续。）

# 解题思路

这是最直观的思路。我们可以想象把三张完全一样的 `Logs` 表并排放在一起。

- **表 1 (L1)**：代表“当前行”。
- **表 2 (L2)**：代表“下一行”（`id` 是 `L1` 的 `id + 1`）。
- **表 3 (L3)**：代表“下下行”（`id` 是 `L1` 的 `id + 2`）。

**逻辑条件：**

1. **ID 连续：**`L2.id = L1.id + 1` 且 `L3.id = L1.id + 2`。
2. **数值相等：**`L1.num = L2.num` 且 `L2.num = L3.num`。

**注意：** 如果数字连续出现了 4 次（例如 1, 1, 1, 1），这个逻辑会匹配到两次（第1-3行，第2-4行）。因此，必须使用 `DISTINCT` 去重。

# SQL代码

```sql
SELECT DISTINCT
    l1.num AS ConsecutiveNums
FROM
    Logs l1
JOIN
    Logs l2 ON l1.id = l2.id - 1
JOIN
    Logs l3 ON l1.id = l3.id - 2
WHERE
    l1.num = l2.num
    AND l2.num = l3.num;
```
